Gestión de Candidatos

Plataforma (backend Django + DRF) para gestión de reclutamiento multiempresa.

Requisitos

Python 3.11+ (probado con 3.13)

Git
### 1️⃣ Clonar el Repositorio

```sh
git clone https://github.com/juandiego305/Gestion-candidatos.git
```

### 2️⃣  Activar  Entorno Virtual(Importante)

```sh
venv\Scripts\activate.bat
```

### 3️⃣ Agregar archivo .env en la raiz del proyecto

El servidor estará disponible en `http://127.0.0.1:8000/`.
